# 介绍
NumPy是使用Python进行科学计算的基础包。它包含如下的内容：

一个强大的N维数组对象。
复杂的（广播）功能。
用于集成C / C ++和Fortran代码的工具。
有用的线性代数，傅里叶变换和随机数功能。
除了明显的科学用途外，NumPy还可以用作通用数据的高效多维容器。可以定义任意数据类型。这使NumPy能够无缝快速地与各种数据库集成。

NumPy是在BSD许可下获得许可的，允许重用而不受限制。
