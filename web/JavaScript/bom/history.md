### history对象

window对象给我们提供了一个 history对象，与浏览器历史记录进行交互。该对象包含用户（在浏览器窗口中）访问过的URL。

| history对象方法 | 作用                                                  |
| ----------------- | ------------------------------------------------------- |
| back()          | 可以后退功能                                          |
| forward()       | 前进功能                                              |
| go(参数)        | 前进后退功能参数如果是1前进1个页面如果是-1后退1个页面 |

history对象一般在实际开发中比较少用，但是会在一些 OA 办公系统中见到。
