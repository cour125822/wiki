## HTML 中有用的字符实体

注释：实体名称对大小写敏感！

| 显示结果 | 描述              | 实体名称     | 实体编号 |
| ---------- | ------------------- | -------------- | ---------- |
|          | 空格              |              |          |
| <        | 小于号            | <            | <        |
| >        | 大于号            | >            | >        |
| &        | 和号              | &            | &        |
| "        | 引号              | "            | "        |
| ’       | 撇号              | ' (IE不支持) | '        |
| ￠       | 分（cent）        | ¢           | ¢       |
| £       | 镑（pound）       | £           | £       |
| ¥       | 元（yen）         | ¥           | ¥       |
| €       | 欧元（euro）      | €           | €       |
| §       | 小节              | §           | §       |
| ©       | 版权（copyright） | ©           | ©       |
| ®       | 注册商标          | ®           | ®       |
| ™       | 商标              | ™           | ™       |
| ×       | 乘号              | ×           | ×       |
| ÷       | 除号              | ÷           | ÷       |

如需完整的实体符号参考，请访问我们的 [HTML 实体符号参考手册](https://www.w3school.com.cn/tags/html_ref_entities.html)。
